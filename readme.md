# Swift-Instruct-Voice

Official repository for our Interspeech 2026 submission:

**Swift-Instruct-Voice: Fast and Flexible Instruction-Guided Bilingual Speech Synthesis**

This repository includes three parts:

- **Dataset/**  
  This folder contains the textual instructions and annotations for the datasets processed in our paper.  
  Since we only release the textual instruction part, links to the original audio sources will also be provided.  
  The complete release is coming soon.

- **Pipeline/**  
  This folder contains prompts and resources used in our instruction generation and instruction rewriting pipelines.  
  Detailed usage instructions are coming soon.

- **Swift-Instruct-Voice/**  
  This folder will contain the code and checkpoints for our TTS model.  
  Coming soon.
